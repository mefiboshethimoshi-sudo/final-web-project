# Quest 1 Submission: Project Plan

**Chosen Project:** Fahari ya Mrembo (Option 5 – Your Own Idea)

## Project Description

Fahari ya Mrembo is an e-commerce website selling stylish bags and pochi.
The site includes 10 products, shopping cart with add/remove functionality, admin product editing, checkout and order history, WhatsApp order integration, search functionality, and a contact form with validation.
The website is responsive and works seamlessly on mobile, tablet, and desktop devices.

## Project Structure

The project consists of the following files and folders:

```
fahari_ya_mrembo/
│
├─ index.html          # Main page with products, cart, checkout, contact
├─ about.html          # About page describing the brand and team
├─ style.css           # Styling for all pages, mobile-first and responsive
├─ script.js           # JavaScript functionality: cart, search, edit, WhatsApp orders, validation
├─ images/             # Folder containing all product images, team image, and favicon
└─ README.md           # Project documentation and instructions
```

## How It Works

1. **Product Catalog:** Users can browse 10 products with name, image, price, and "Add to Cart" buttons.
2. **Cart System:** Users can add products to the cart, increase/decrease quantity, remove items, and view total price.
3. **Admin Features:** Admin can edit product names and prices after entering a password.
4. **Search Functionality:** Users can search products in real-time.
5. **Checkout & Orders:** Users can confirm orders, view order history, and send orders via WhatsApp.
6. **Contact Form:** Users can submit messages, with input validation to ensure proper entries.
7. **Responsive Design:** Layout adjusts for mobile, tablet, and desktop devices using media queries.
8. **Polished UI:** Proper styling, headings, buttons, and accessibility features like alt text for images.
