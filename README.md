# Fahari ya Mrembo 👜

## Project Overview
Fahari ya Mrembo is a fully functional e-commerce website designed to sell stylish bags and pochi. It demonstrates HTML, CSS, and JavaScript skills.

## Features
- 10 products display
- Add to cart, remove, clear cart
- Checkout and order history
- WhatsApp order with total price
- Admin product editing
- Search functionality
- Contact form validation
- Responsive design

## Project Structure
index.html
about.html
style.css
script.js
images/
README.md

## How to Run
Open index.html in a browser.
